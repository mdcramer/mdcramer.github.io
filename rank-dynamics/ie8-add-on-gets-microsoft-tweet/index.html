    IE8 Add-on Gets Microsoft Tweet

    We made our application compatible with IE8 back in March with the v2.0.3 release, and then enhanced the IE8 functionality in July with the v3.0.0 release by taking advantage of some specific IE8 features. In particular, we included image previews, along with the top 3 search results, directly in the SearchBar drop-down.

    The official twitter page for Internet Explorer recognized our work this morning with the following tweet, which links to our add-on in the Internet Explorer Add-ons Gallery:

    “Semantic real-time implicit personalization with Surf Canyon! What more could you want out of an IE8 add-on? http://bit.ly/kphrm”

    Also in the IE Gallery you can find our Instant Visual Search plug-in, which offers just the SearchBar integration, as well as the Search Assistant by OneRiot. Together with OneRiot, the real-time search engine company with whom we’ve worked in the past, we’ve developed a OneRiot-customized version of the IE add-on for those users who wants to make sure to get real-time results when they’re searching on Bing, Google or Yahoo!:
